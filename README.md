# WaveFucker by BonarSound

- [x] Naive synthesis method implementation
- [x] Oscilloscope
- [x] Spectrum Analyzer (FFT)
- [x] BLIT (Band-Limited Impulse Train) method implementation

- [ ] DPW method implementation
